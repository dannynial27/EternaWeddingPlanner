# Eterna | Luxury Wedding & Event Planning

![Eterna Logo](https://images.unsplash.com/photo-1519741497674-611481863552?auto=format&fit=crop&q=80&w=1000)

Eterna is an ultra-premium, "bombastic" front-end template designed for high-end wedding planning businesses. It features an exquisite design language incorporating modern luxury trends, glassmorphism, fluid animations, and a rich aesthetic tailored for the Malaysian market.

## ✨ Features

- **Ultra-Premium Aesthetic:** A refined dark-and-gold color palette with delicate typography.
- **Tactile Noise Overlay:** A subtle grain texture applied across the site for a physical, high-end print feel.
- **Gold Foil Effects:** Key headings feature a shimmering, animated metallic gold gradient.
- **Magnetic Buttons:** Interactive buttons that subtly track the user's cursor.
- **Glassmorphism:** Elegant frosted glass pricing cards and navigation elements.
- **Canvas Particle System:** Floating light "dust" particles enhancing the hero section.
- **Custom Trailing Cursor:** A sleek dot and trailing outline that replaces the default browser cursor.
- **Parallax Backgrounds:** Smooth depth effects on scroll.
- **Staggered Scroll Reveals:** Content uniquely animates into view as the user navigates.
- **Localized Context:** Pricing reflected in RM and localized testimonials/venues (Kuala Lumpur, Langkawi, Penang).

## 🛠️ Built With

- **HTML5:** Semantic structuring.
- **CSS3:** Advanced styling, CSS variables, Keyframe animations, and Flexbox/Grid layouts.
- **Vanilla JavaScript:** Zero-dependency logic handling scroll events, canvas particles, cursor tracking, and intersection observers.
- **Phosphor Icons:** For highly legible and modern iconography.
- **Google Fonts:** Playfair Display and Lato.

## 🚀 Getting Started

This project is a static front-end build and requires no complex setup.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/dannynial27/EternaWeddingPlanner.git
   ```
2. **Open the project:**
   Simply open `index.html` in any modern web browser to view the application.

## 📱 Responsiveness

The project is fully responsive across desktop, tablet, and mobile breakpoints. Note that certain ultra-premium features (like the custom trailing cursor and magnetic buttons) are disabled on screens smaller than 992px to ensure an optimal touch experience.
